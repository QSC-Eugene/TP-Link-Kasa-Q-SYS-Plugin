# TP-Link Kasa Plugin

*Warning: I have only verified this plugin in my own enviroment and needs community testing.*

Tested on QDS 9.12.1 with a Core 110f

## Functionality

- All
  - Device Name (Read/Write)
  - Mac Address (Read Only)
- Switches
  - On/Off
  - Toggle
- Dimmers
  - On/Off
  - Toggle
  - Brightness

## Kasa Devices Tested

- KS200M
- EP10
- EP40
- HS220
- HS103
- HS200
- HS300
- KP115
- KS230
- KL400L5
- KL420L5
